Chapter 03

Core Concepts

Understanding the architecture behind Seaborn makes you dramatically more effective.

Figure-Level vs Axes-Level Functions

This is the most important distinction in Seaborn. Every function is either figure-level or axes-level:

Figure-Level Functions
  • Return a FacetGrid object
  • Manage their own figure and subplots
  • Support col= and row= faceting
  • Examples: relplot, displot, catplot, lmplot
Axes-Level Functions
  • Return a matplotlib.Axes object
  • Draw into existing axes
  • Composable with matplotlib subplots
  • Examples: scatterplot, histplot, boxplot
python — figure-level vs axes-level
# FIGURE-LEVEL — manages its own figure
g = sns.relplot(data=tips, x="total_bill", y="tip", col="time")
g.set_axis_labels("Total Bill ($)", "Tip ($)")
g.set_titles("{col_name} Dining")

# AXES-LEVEL — works inside your subplot layout
fig, axes = plt.subplots(1, 2, figsize=(12, 5))
sns.scatterplot(data=tips, x="total_bill", y="tip", ax=axes[0])
sns.histplot(data=tips, x="total_bill", ax=axes[1])
plt.tight_layout()
plt.show()

The Data Specification System

python
tips = sns.load_dataset("tips")

# Seaborn accepts data in multiple forms:

# 1. DataFrame + column names (preferred in industry)
sns.scatterplot(data=tips, x="total_bill", y="tip", hue="sex")

# 2. Long-form DataFrame (tidy data format)
df_long = tips.melt(id_vars=["sex"], value_vars=["total_bill", "tip"])
sns.boxplot(data=df_long, x="variable", y="value", hue="sex")

# 3. Wide-form DataFrame (for corr matrices etc.)
penguins = sns.load_dataset("penguins")
sns.heatmap(penguins.select_dtypes("number").corr(), annot=True)
Industry Best Practice

Always use tidy (long-form) data. One observation per row, one variable per column. Seaborn's semantic mapping (hue, size, style) works perfectly with this format.

    y="signal",

relplot — the figure-level gateway

python
# relplot is figure-level: wraps scatterplot or lineplot
# Use it when you need col= or row= faceting

g = sns.relplot(
    data=tips,
    x="total_bill",
    y="tip",
    col="time",           # Creates separate subplots per "time" value
    hue="smoker",
    style="smoker",
    kind="scatter",       # or "line"

    aspect=0.9

When to use relplot vs scatterplot

Use scatterplot/lineplot when composing with matplotlib subplots. Use relplot when you want faceting or a standalone figure.

histplot — modern replacement for distplot

python
# histplot replaced the deprecated distplot() in v0.11

sns.histplot(
    data=penguins, x="flipper_length_mm",

    ax=axes[1]

sns.histplot(
    data=penguins, x="flipper_length_mm",
    y="body_mass_g", bins=30,
    cbar=True, ax=axes[2]
)

plt.tight_layout()
plt.show()

    data=penguins,
    x="flipper_length_mm",
    y="body_mass_g",

)

Chapter 09

Matrix Plots

Essential for correlation analysis and exploring tabular data patterns.

heatmap — correlation matrices and beyond

python
penguins = sns.load_dataset("penguins")

# Correlation matrix heatmap
numeric_cols = penguins.select_dtypes(include="number")
corr_matrix = numeric_cols.corr()

# Create mask for upper triangle (avoid redundancy)
mask = np.triu(np.ones_like(corr_matrix, dtype=bool))

fig, ax = plt.subplots(figsize=(8, 6))
sns.heatmap(
    corr_matrix,
    mask=mask,                  # Show only lower triangle
    annot=True,               # Show values in cells
    fmt=".2f",                 # Format annotation values
    cmap="coolwarm",           # Diverging colormap (best for correlation)
    center=0,                  # Center colormap at 0
    vmin=-1, vmax=1,
    square=True,
    linewidths=0.5,
    cbar_kws={"shrink": 0.75},
    ax=ax
)
ax.set_title("Feature Correlation Matrix", pad=20)
plt.tight_layout()
Colormap Selection Matters

Use diverging colormaps (coolwarm, RdBu, PiYG) for data centered around 0. Use sequential colormaps (YlOrRd, viridis, Blues) for unidirectional data. Avoid jet — it's perceptually misleading.

clustermap — hierarchical clustering

python
# clustermap reorders rows/columns by hierarchical clustering
flights = sns.load_dataset("flights")
pivot = flights.pivot_table(
    index="month", columns="year", values="passengers"
)

g = sns.clustermap(
    pivot,
    cmap="YlOrRd",
    standard_scale=1,         # Normalize within rows (0 to 1)
    figsize=(12, 8),
    linewidths=0.1,
    annot=True,
    fmt=".0f",
    col_cluster=False,        # Don't cluster columns (time is ordered)
    row_cluster=True          # Cluster months
)
g.fig.suptitle("Airline Passengers Heatmap", y=1.02)

pairplot — multivariate exploration

FacetGrid — full control over multi-plot layout

python
# FacetGrid gives maximum flexibility for custom multi-panel figures
tips = sns.load_dataset("tips")

g = sns.FacetGrid(
    tips,
    col="time",
    row="smoker",
    hue="sex",
    height=4,
    aspect=1,
    margin_titles=True    # Titles on margin, not each cell
)

g.map(sns.scatterplot, "total_bill", "tip", alpha=0.7)
g.add_legend()
g.set_axis_labels("Total Bill ($)", "Tip ($)")
g.set_titles(col_template="{col_name}", row_template="Smoker: {row_name}")

# Apply custom function across all axes
def add_ref_line(*args, **kwargs):
    plt.axhline(y=5, color="red", linestyle="--", alpha=0.5, label="$5 tip")

g.map(add_ref_line)
plt.show()

PairGrid — custom pair plots

python
# PairGrid lets you specify different plot types per region
g = sns.PairGrid(penguins.dropna(), hue="species",
                  vars=["bill_length_mm", "flipper_length_mm", "body_mass_g"])

g.map_upper(sns.scatterplot, alpha=0.6)   # Upper triangle: scatter
g.map_lower(sns.kdeplot, fill=True)       # Lower triangle: KDE contours
g.map_diag(sns.histplot, kde=True)       # Diagonal: histogram + KDE
g.add_legend()

Step 4 — Feature Correlation with Significance

python
from scipy import stats

# Compute correlation and p-values
numeric_titanic = titanic.select_dtypes(include="number").dropna()
corr = numeric_titanic.corr()

# Compute p-value matrix
def compute_pvalues(df):
    p_matrix = pd.DataFrame(np.ones((df.shape[1], df.shape[1])),
                              columns=df.columns, index=df.columns)
    for col1 in df.columns:
        for col2 in df.columns:
            _, p = stats.pearsonr(df[col1], df[col2])
            p_matrix.loc[col1, col2] = p
    return p_matrix

p_vals = compute_pvalues(numeric_titanic)
sig_mask = p_vals > 0.05   # Mask non-significant correlations

sns.heatmap(corr, annot=True, fmt=".2f", cmap="coolwarm",
           center=0, mask=sig_mask, linewidths=0.5,
           cbar_kws={"label": "Pearson r (p < 0.05)"})
plt.title("Significant Correlations Only")
Chapter 13

Common Mistakes & Anti-Patterns

Avoid these pitfalls that trip up even experienced analysts.

Correct
python
# Use dropna() before certain plots
sns.pairplot(penguins.dropna())

# Use displot instead of distplot
sns.displot(x="total_bill", data=tips)

# Call tight_layout before show/savefig
plt.tight_layout()
plt.show()
Anti-Pattern
python
# NaN causes cryptic errors in pairplot
sns.pairplot(penguins)  # ❌

# distplot is deprecated since v0.11
sns.distplot(tips["total_bill"])  # ❌

# Clipped labels / overlapping text
plt.show()  # ❌ (no tight_layout)

Anti-patterns checklist

Never use distplot()
Deprecated since v0.11. Replace with histplot(kde=True) or displot().
Don't mix figure-level and axes-level
Don't pass ax= to relplot, displot, or catplot — they manage their own figures.
Respect hue_order and order
Always use order= and hue_order= for categorical plots to prevent randomized legend/axis ordering across runs.
Don't use pie charts
Seaborn intentionally doesn't support pie charts. Use barplot or countplot — they're more readable.
Always save with bbox_inches="tight"
Otherwise legends and labels get cropped in the saved file even if they look fine on screen.

Saving Publication-Quality Figures

python
# For papers / reports (300 DPI, tight bounding box)
plt.savefig("figure.png", dpi=300, bbox_inches="tight",
           facecolor="white", transparent=False)

# Vector format for infinite zoom (presentations, web)
plt.savefig("figure.svg", bbox_inches="tight")
plt.savefig("figure.pdf", bbox_inches="tight")  # LaTeX papers

# Set global DPI for all figures
plt.rcParams["savefig.dpi"] = 300
plt.rcParams["savefig.bbox"] = "tight"
